✔️ = Complete.

⌨️ = BBCode dropped in favor of HTML equivalent.

🎉 = Powered by official Discourse Addon.

☠️ = Do not proceed. Markdown-it and/or Discourse do not like this code. Use Markdown equivalent. Unable to be rebaked.
